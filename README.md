# 个人主页 - Personal Homepage

一个基于 GTA5 武器轮盘风格的个人主页，使用 React + TypeScript + Vite 构建。

## ✨ 特性

- 🎯 **GTA5 风格轮盘菜单** - 鼠标悬停中心圆圈展开四个功能按钮
- 📝 **可编辑内容** - 所有内容都可以在页面上直接编辑
- 🎨 **现代灰色调设计** - 高级灰白配色方案
- 📱 **响应式布局** - 适配各种屏幕尺寸
- ⚡ **流畅动画** - 使用 Framer Motion 实现丝滑动画效果

## 🚀 功能模块

1. **知识库** - 树状图展示知识体系
2. **个人项目** - 展示个人项目作品
3. **实习/工作** - 简历信息展示（点击中心圆圈打开）
4. **个人博客** - 博客文章列表

## 📦 技术栈

- React 18.3.1
- TypeScript 5.6.2
- Vite 6.0.1
- Tailwind CSS 3.4.15
- Framer Motion 12.40.0
- React Router DOM 6.28.0
- lucide-react 1.21.0

## 🛠️ 安装运行

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 📝 使用方法

1. 启动开发服务器：`npm run dev`
2. 在浏览器中打开：`http://localhost:5173`
3. 鼠标悬停中心圆圈查看轮盘菜单
4. 点击中心圆圈查看简历
5. 点击各功能按钮进入对应页面
6. 所有文本内容都可以直接点击编辑
